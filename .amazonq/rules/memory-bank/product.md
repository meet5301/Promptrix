# Promptrix — Product Overview

## Purpose
Promptrix is an AI-powered website builder that uses Claude (Anthropic) to generate complete, multi-page HTML websites from natural language prompts. Users describe what they want, and the app produces ready-to-download websites.

## Value Proposition
- Zero-code website creation via natural language
- Template-based customization for structured, professional results
- Instant preview and ZIP download of generated sites
- Iterative refinement — modify generated sites with follow-up prompts

## Key Features
- **LLM Generation** — Claude generates full HTML/CSS/JS websites from prompts
- **4 Templates** — Modern, Creative, Minimal, Professional with deep customization
- **Template Customization** — Header, hero, home sections, footer, colors, page count
- **Real-time Preview** — Live iframe with responsive viewport switching
- **Refine Mode** — Follow-up instructions to modify an already-generated site
- **Download** — ZIP archive of all generated HTML pages
- **User Auth** — Login/register with session management
- **Site History** — MongoDB-backed persistence of generated sites per user

## Target Users
- Non-developers who need quick websites
- Designers prototyping layouts
- Small businesses wanting a fast web presence

## Use Cases
- Portfolio sites, landing pages, e-commerce storefronts, SaaS marketing pages
- Rapid prototyping from a text description
- Template-driven branded sites with custom colors and content
